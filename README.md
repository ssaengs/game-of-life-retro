# Game of Life - Pixel Art (Node.js)

A web-based Game of Life with a pixel-art style UI, served by a Node.js Express server.

## How to Run

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the server:**
   ```bash
   npm start
   ```

3. **Open your browser:**
   Go to [http://localhost:3000](http://localhost:3000)

4. **Play!**
   - Click cells to toggle them.
   - Use the controls to start, pause, clear, randomize, and adjust speed.
